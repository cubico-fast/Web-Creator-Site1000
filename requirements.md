## Packages
@dnd-kit/core | Core drag and drop primitives
@dnd-kit/sortable | Sortable presets for drag and drop lists
@dnd-kit/utilities | Utilities for drag and drop
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind classes efficiently

## Notes
Authentication is handled via Replit Auth (useAuth hook).
Editor uses a simplified block-based JSON structure stored in the 'content' column of the 'pages' table.
Images use Unsplash source URLs for demonstration.
